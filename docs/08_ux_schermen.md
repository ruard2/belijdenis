# UX Schermen

## Leerling-app

### Start

Toont:
- actieve cursus
- voortgang
- volgende les
- recente groepsvragen

### Cursussen

Toont:
- beschikbare cursussen
- voortgang per cursus
- lock/status indien niet beschikbaar

### Hoofdstukken

Toont:
- hoofdstukken in volgorde
- status per hoofdstuk
- XP
- voltooid/nog te doen

### Lesson player

Toont blocks een voor een of als verticale flow.

MVP-advies:
- verticale flow is sneller te bouwen
- per block duidelijke call-to-action
- sticky voortgangsbalk

### Community bij block

Toont:
- ontdekkingen uit eigen groep
- reactieknop
- vraagknop
- eigen bijdrage

### Profiel

Toont:
- naam
- groep(en)
- XP
- badges
- voortgang

## Admin-app

### Dashboard

Toont:
- cursussen
- groepen
- open vragen
- concepten
- moderatiemeldingen

### Cursus editor

Toont:
- metadata
- hoofdstukkenlijst
- status
- acties

### Hoofdstuk editor

Toont:
- titel/subtitel
- XP
- status
- blocks in volgorde
- previewknop
- publiceerknop

### Block editor

Toont op basis van blocktype:
- velden
- validatiefouten
- preview
- XP
- required toggle

## Designrichting

Professioneel, rustig en warm.

Niet:
- te kinderlijk
- drukke game-interface
- overdreven marketingstijl

Wel:
- duidelijke navigatie
- zachte progressie
- goed leesbare tekst
- focus op groepsgesprek
- visueel rijk waar inhoud daarom vraagt
