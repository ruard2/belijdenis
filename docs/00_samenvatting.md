# Samenvatting

Houvast is een app voor catechese, geloofsvorming en groepsgesprek.

De app moet jongeren en groepen helpen om geloof niet alleen te consumeren, maar te ontdekken, te bespreken, te verdiepen en persoonlijk te maken.

## Productbelofte

Houvast maakt catechese interactief, persoonlijk en schaalbaar:
- catecheten beheren inhoud zonder programmeur
- leerlingen volgen lessen in korte blocks
- groepen delen ontdekkingen en vragen
- voortgang, XP en badges maken groei zichtbaar
- de engine kan later ook andere cursussen dragen

## Belangrijkste ontwerpkeuze

Content is data.

De app is geen verzameling vaste schermen per hoofdstuk. De app is een engine die gepubliceerde content uit de database rendert.

## MVP

De MVP bevat:
- Flutter leerling-app
- FastAPI backend
- PostgreSQL database
- Railway deployment
- authenticatie
- rollen
- groepen
- content-driven lesson engine
- admin-CMS
- draft/published workflow
- voortgang
- XP-events
- ontdekkingen
- vragen bij blocks
- seed-content voor hoofdstuk 1 en 2

## Niet in MVP

Bewust later:
- notificaties
- uitgebreide analytics
- ouder/verzorgerportaal
- offline mode
- AI-functies
- betalingen/licenties
- multi-tenant kerkenstructuur

## Succesdefinitie

Het project is goed opgezet als een catecheet zelfstandig:
- een cursus kan maken
- hoofdstukken kan toevoegen
- blocks kan beheren
- content kan publiceren
- leerlingen in groepen kan laten werken
- reacties en vragen kan begeleiden
